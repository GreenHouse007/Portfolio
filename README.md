## 🧰 Getting Started (Local Development)

Clone and install dependencies:

```bash
git clone https://github.com/GreenHouse007/my-portfolio.git
cd my-portfolio
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view.

---

## 🧾 Environment Variables

Create a `.env.local` file in the root directory with the following:

```bash
RESEND_API_KEY=your_resend_api_key
RESEND_FROM="Alexander Green <yourname@resend.dev>"
CONTACT_TO=youremail@example.com
```
